Rules

How to play:

  • Press Kick-off to start the game
  • Each question will have four possible answers, select one by clicking on it
  • If the correct answer has been selected it will turn green, if not the selected answer will turn red
  • The next question will load automatically
  • Once complete, you will be presented with your final score
  • You can enter your name to save your score to the league table
  • You will then be brought back to the homepage where you can view the league table or play the quiz again!
